## **Описание проекта**

Проект представляет собой систему интеллектуального поиска по видеоматериалам с использованием Retrieval-Augmented Generation (RAG).
 Источник данных — образовательные видеоролики с YouTube. Для каждого видео выполняется автоматическая транскрибация, очистка текста, разбиение на смысловые сегменты и построение индекса для последующего поиска по содержанию.

Пользователь задаёт вопрос, после чего система:

1. выполняет поиск релевантных фрагментов среди подготовленных чанков;
2. извлекает наиболее значимые сегменты;
3. формирует итоговый ответ с использованием LLM.

Система поддерживает локальный запуск и интеграцию с ботом/веб-интерфейсом.

------

# **Инструкция по запуску**

### **1. Создать виртуальное окружение**

```
python3 -m venv venv
source .venv/bin/activate
```

### **2. Установить зависимости**

```
pip install -r requirements.txt
```

### **3. Запустить RAG-приложение**

```
python app/app.py
```

------

# **Использование**

После запуска система принимает запросы от пользователя и возвращает ответ, основанный на извлечённых сегментах видео.

Примеры запросов:

```
"В чём основная идея курса?"
"Что такое ML?"
```
